# MIP Search Engine

Αυτό το εργαλείο επιτρέπει την αναζήτηση λέξεων-κλειδιών σε αρχεία .txt που προέρχονται από την ιστοσελίδα του Υπουργείου Εσωτερικών (mip.gov.cy).

## Τεχνολογίες
- Python
- Flask

## Πώς να το τρέξετε
1. Τοποθετήστε τα .txt αρχεία σας στον φάκελο `mip_texts`
2. Τρέξτε την εφαρμογή με:
   ```
   python mip_search_app.py
   ```
3. Ανοίξτε το πρόγραμμα περιήγησης στο:
   ```
   http://localhost:8000
   ```

## Παράδειγμα αναζήτησης
Αναζητώντας την φράση: `άδεια παραμονής` εμφανίζονται τα σχετικά αποσπάσματα από όλα τα αρχεία.
